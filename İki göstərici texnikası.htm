İki göstərici texnikası

İki göstərici texnikası, kompüter elmləri və proqramlaşdırmada massivlər və ya ardıcıllıqlarla bağlı problemlərin həllində geniş istifadə olunur. Bu metod, bir massiv və ya siyahı üzərində eyni anda iki fərqli mövqedə duran göstəricilərlə (index-lərlə) işləməyi nəzərdə tutur. Göstəricilər bəzən əks istiqamətlərdə, bəzən isə fərqli sürətlə hərəkət edir.

Bu texnika adətən aşağıdakı hallarda tətbiq olunur:

  • Massivdə müəyyən cüt elementləri tapmaq üçün (məsələn, cəmi müəyyən bir ədədə bərabər olan cütlər);
  • Massivi və ya siyahını tərsinə çevirmək üçün;
  • İki siyahını birləşdirmək və ya sıralı massivlər üzərində axtarış aparmaq üçün.

İş prinsipi

  1. Başlanğıc: Göstəricilər massivdə müxtəlif mövqelərə yerləşdirilir (məsələn, biri əvvəlində, biri sonunda).
  2. Hərəkət: Müəyyən şərtlərə əsasən göstəricilər hərəkət edir.
  3. Yoxlama: Hər addımda şərtlər yoxlanılır, məsələn cütlərin cəmi, sıralama və s.
  4. Tamamlanma: Şərtlər ödənmədikdə və ya göstəricilər görüşdükdə proses dayanır.

Python realizasiyası — siyahının tərsinə çevrilməsi

İndi iki göstərici texnikasını ən sadə nümunə ilə izah edək — siyahını tərsinə çevirmək. Bunun üçün biri başlanğıcdan, digəri sondan başlayan iki göstəricidən istifadə edəcəyik.


    # Siyahının uzunluğunu və elementlərini oxuyuruq
    n = int(input("Elementlərin sayını daxil edin: "))
    lst = list(map(int, input("Elementləri daxil edin: ").split()))
    
    # İki göstərici müəyyən edirik
    sol = 0
    sag = n - 1
    
    # Göstəricilər görüşənə qədər dəyişdiririk
    while sol < sag:
        lst[sol], lst[sag] = lst[sag], lst[sol]
        sol += 1
        sag -= 1
    
    print("Tərsinə çevrilmiş siyahı:")
    print(*lst)
İzah:
  • Əvvəlcə siyahının elementlərini oxuyuruq.
  • sol göstərici 0-dan, sag göstərici isə son elementdən başlayır.
  • Hər döngədə bu iki elementin yerləri dəyişdirilir.
  • Beləcə siyahı tərsinə çevrilmiş olur.

Python realizasiyası — cəmi müəyyən ədədə bərabər olan cütlərin tapılması

Bu nümunədə iki göstərici texnikasından istifadə edərək, cəmi target-ə bərabər olan element cütlərini tapacağıq.


    # Giriş məlumatlarını oxuyuruq
    n = int(input("Elementlərin sayını daxil edin: "))
    lst = list(map(int, input("Elementləri daxil edin: ").split()))
    target = int(input("Axtarılan cəmi daxil edin: "))
    
    # Siyahını sıralayırıq
    lst.sort()
    
    # Göstəricilər
    sol = 0
    sag = n - 1
    
    # Cütləri tapırıq
    while sol < sag:
        cem = lst[sol] + lst[sag]
        if cem == target:
            print(lst[sol], "+", lst[sag], "=", target)
            sol += 1
            sag -= 1
        elif cem < target:
            sol += 1
        else:
            sag -= 1
İzah:
  • Siyahı əvvəlcə sıralanır.
  • sol və sag göstəriciləri ilə başlanır.
  • Əgər cəmləri hədəf ədədə bərabərdirsə, nəticə çap olunur.
  • Cəm hədəfdən kiçikdirsə, sol göstərici artırılır, böyükdürsə, sag azaldılır.
Nəticə: İki göstərici texnikası kodun sadələşdirilməsinə, vaxtın optimallaşdırılmasına və iç-içə döngələrin qarşısının alınmasına kömək edir. Bu üsul çox vaxt O(n) mürəkkəbliklə işləyir və səmərəli həll təqdim edir.
